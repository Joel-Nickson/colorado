# Install

1.  Clone:

        git clone https://github.com/NabeelAhmed1721/colorado.git && cd colorado

2.  Install:

        yarn install

    or

        npm install

3.  Compile:

        yarn build

    or

        npm run build

4.  Add Chrome Extension by loading unpacked and selecting `dist` project in project directory.

5.  Enjoy :)
